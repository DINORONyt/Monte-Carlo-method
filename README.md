# Лабораторная работа №2: Разработка приложений с графическим интерфейсом Windows Forms. Условия и циклы
## Вариант 17: Проверка попадания точки и метод Монте-Карло

### Описание проекта
Данный проект представляет собой приложение на языке C# с графическим интерфейсом Windows Forms, разработанное в рамках лабораторной работы №2 по дисциплине "Алгоритмизация и программирование". Приложение реализует три задачи, связанные с геометрическими вычислениями, методом Монте-Карло и визуализацией данных через класс `Graphics`. Бизнес-логика вынесена в отдельную библиотеку классов, интерфейс и код разделены, проект покрыт unit-тестами.

### Функциональность
**Задача 1 (Базовая)** — Проверка попадания точки в круг  
Пользователь вводит координаты центра круга, радиус и координаты проверяемой точки. Программа определяет положение точки: внутри, на границе или снаружи.

**Задача 2 (Средняя)** — Генерация случайных точек и оценка числа π  
Генерируется N случайных точек в единичном квадрате. Подсчитывается количество точек, попавших в вписанную окружность. На основе соотношения вычисляется приближённое значение числа π.

**Задача 3 (Расширенная)** — Визуализация точек и графический вывод π  
В компоненте `PictureBox` отрисовываются квадрат, вписанная окружность и сгенерированные точки. Точки внутри круга окрашиваются в зелёный цвет, снаружи — в красный. Внизу отображается текущее вычисленное приближение π.

Использование приложения
При запуске открывается окно с тремя вкладками, соответствующими задачам:
Вкладка 1: Проверка точки
Введите координаты центра круга (X, Y) и радиус
Введите координаты проверяемой точки
Нажмите кнопку Проверить
Результат отобразится под кнопкой
Вкладка 2: Оценка π
Введите количество точек N (рекомендуется от 1000 до 1000000)
Нажмите Сгенерировать точки
Программа выведет приближённое значение π и статистику попаданий
Вкладка 3: Визуализация
Нажмите Обновить или перейдите на вкладку после генерации
В PictureBox отобразятся: квадрат, окружность и точки
Внизу отобразится актуальное значение π на основе отрисованных точек
Примеры работы
Задача 1:
Ввод: Центр (0; 0), Радиус = 5, Точка (3; 4)
Результат: Точка находится внутри круга (расстояние = 5, но с учётом эпсилона может быть определено как на границе, в зависимости от точности)
Задача 2:
Ввод: N = 10000
Результат: Оценённое π: 3.142400 | Попало в круг: 7856 из 10000
Задача 3:
Визуализация показывает плотное распределение точек, зелёные внутри круга, красные снаружи. График подтверждает сходимость к ~3.14.
Обработка ошибок
Приложение включает валидацию ввода на уровне формы и логики:
Проверка на пустые поля ввода
Проверка формата чисел (поддержка десятичного разделителя .)
Запрет отрицательного или нулевого радиуса
Ограничение диапазона N: от 1 до 1 000 000
Вывод сообщений об ошибках через MessageBox с иконкой предупреждения
Архитектурная защита: методы валидации вынесены в TaskSolver, что предотвращает дублирование кода
